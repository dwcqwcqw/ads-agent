## 七、设计原则

### 1. 图片策略
- ✅ **占位符图标**：避免网络图片加载失败
- ✅ **统一风格**：紫色渐变背景 + Emoji图标
- ✅ **大小适中**：Agent 3用4rem，Agent 4用5rem

### 2. 对应关系
- ✅ **广告主一致**：Agent 3和Agent 4使用相同的广告主
- ✅ **ID映射**：通过adId字段建立对应关系
- ✅ **图标关联**：相关联的图片使用相近或相同的图标

### 3. Banner设计
- ✅ **左图右文**：符合阅读习惯
- ✅ **信息层次**：标题 > 副标题 > 正文 > CTA
- ✅ **视觉焦点**：推荐广告突出显示

### 4. 响应式布局
- ✅ **自适应列数**：Agent 3使用CSS Grid
- ✅ **固定比例**：Banner图片保持200x150比例
- ✅ **移动端适配**：小屏幕下垂直堆叠

## 十一、故障排查

### 问题1：Banner图标不显示
**症状**：部分Emoji图标在某些设备上显示为方框

**解决方案**：
```javascript
// 使用系统字体支持的表情符号
<div style={{fontFamily: 'Apple Color Emoji, Segoe UI Emoji, Noto Color Emoji, sans-serif'}}>
  {banner.image}
</div>
```

### 问题2：Banner布局错乱
**症状**：左图右文布局在小屏幕上换行混乱

**解决方案**：
```javascript
// 使用flex-wrap处理响应式
<div style={{
  display: 'flex',
  flexWrap: 'wrap',  // 自动换行
  gap: '1rem'
}}>
  <div style={{flex: '1 1 200px'}}>{/* 图片 */}</div>
  <div style={{flex: '2 1 300px'}}>{/* 文案 */}</div>
</div>
```

### 问题3：图标大小不一致
**症状**：不同Emoji图标大小差异明显

**解决方案**：
```javascript
// 使用固定尺寸的SVG图标替代Emoji
const icons = {
  '📚': <BookIcon />,
  '🎓': <GraduationIcon />,
  // ...
}
```
