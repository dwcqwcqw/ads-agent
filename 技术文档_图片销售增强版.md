## 一、图片生成问题修复

### 问题描述
Agent 4 广告创意生成时，AI图片生成失败，无法展示广告图片。

### 原因分析
原代码使用 `gpt-image-1.5` 模型，该模型存在兼容性问题或未在OpenAI API中正式发布。

### 解决方案
切换为成熟的 `dall-e-3` 模型：

**修改位置**：`ad_system.py` 第489行

**修改前**：
```python
image_response = client.images.generate(
    model="gpt-image-1.5",  # ❌ 不稳定的模型
    prompt=image_prompt,
    size="1024x1024",
    quality="high",
    n=1
)
```

**修改后**：
```python
image_response = client.images.generate(
    model="dall-e-3",  # ✅ 稳定的生产级模型
    prompt=image_prompt,
    size="1024x1024",
    quality="standard",
    n=1
)
```

### 参数说明
| 参数 | 值 | 说明 |
|------|-----|------|
| model | dall-e-3 | OpenAI官方图片生成模型 |
| size | 1024x1024 | 正方形图片尺寸 |
| quality | standard | 标准质量（平衡速度与效果）|
| n | 1 | 生成1张图片 |

### 备选方案
考虑到API调用可能失败，系统实现了降级处理：

```python
except Exception as e:
    console.print(f"[red]✗[/red] 图片生成失败: {str(e)}")
    image_path = ""
    creative["generated_image_url"] = ""  # 失败时返回空字符串
```

### 销售风格说明

#### 咨询式销售（Consultative Selling）
**核心理念**：
- 不是推销产品，而是提供解决方案
- 先理解客户，再推荐产品
- 建立信任是成交的基础

**适用场景**：
- 高单价产品（课程2999-4999元）
- 教育、医疗等专业领域
- 企业管理者等高学历客户

**沟通步骤**：
1. **破冰** (0-30秒)：建立友好关系
2. **需求挖掘** (30秒-3分钟)：通过提问了解痛点
3. **痛点共鸣** (3-5分钟)：认同客户焦虑
4. **方案展示** (5-10分钟)：针对性介绍产品
5. **信任建立** (10-15分钟)：展示案例和口碑
6. **促成** (15分钟+)：处理异议，达成成交

### 销售技能库

#### 技能1：需求挖掘（Needs Discovery）

**定义**：通过开放式问题深入了解客户的具体需求、痛点和期望。

**核心问题框架**：
```
SPIN模型：
- S (Situation)：现状问题
  "孩子现在几年级？"
  "平时数学成绩怎么样？"
  
- P (Problem)：困难问题
  "做应用题时主要遇到什么困难？"
  "哪些类型的题目最让孩子头疼？"
  
- I (Implication)：影响问题
  "这个问题对孩子的考试成绩影响大吗？"
  "会不会影响到小升初？"
  
- N (Need-payoff)：价值问题
  "如果解决了这个问题，对孩子有什么好处？"
```

**在本案例中的应用**：
```javascript
// 对话示例
AI: "请问孩子做应用题时，主要是读不懂题意，还是能读懂但不知道怎么列式计算呢？"
User: "主要是读完题不知道怎么列式，尤其是行程问题、工程问题，一看到就懵。"
AI: "我明白了！这其实是典型的数量关系建模能力不足。"
```

#### 技能2：痛点共鸣（Pain Points）

**定义**：认同客户的焦虑和困扰，让客户感受到被理解和被重视。

**共鸣技巧**：
```
技巧1：镜像法
"我完全理解您的担忧。很多家长都有同感。"

技巧2：案例分享
"之前有个深圳南山区的小孩，也是五年级，也是应用题困难..."

技巧3：情感确认
"应用题确实是小学数学的分水岭，很多孩子都在这里遇到瓶颈。"
```

**注意事项**：
- ❌ 不要过度夸大问题，制造焦虑
- ❌ 不要贬低客户之前的决定
- ✅ 真诚、共情、不过度推销

#### 技能3：方案展示（Solution Presentation）

**定义**：根据客户需求，有针对性地介绍产品功能和优势。

**展示框架**：
```
FAB法则：
- F (Feature)：特性
  "我们的课程采用清华名师团队研发的审题四步法"
  
- A (Advantage)：优势
  "这个方法经过10万+学员验证，平均提分15分以上"
  
- B (Benefit)：利益
  "您的孩子也能在短时间内掌握这个方法，下次考试应用题不再丢分"
```

**个性化展示**：
```javascript
// 针对本案例（企业高管家庭）
"我们特别适合您这种情况：
- 家长工作忙，孩子需要高效的学习方法
- 您注重品质和效果，我们的清华名师正是为这种家庭设计的
- 0元试听让您先看效果再决定，完全没有风险"
```

#### 技能4：信任建立（Trust Building）

**定义**：通过展示权威性、社会证明和可信度，消除客户的购买顾虑。

**信任要素**：
```
1. 权威背书
   - "清华名师团队"
   - "10年教学经验"
   - "深圳多所重点中小学指定辅导机构"

2. 社会证明
   - "已帮助超过10万学生提升成绩"
   - "95%的家长推荐"
   - "深圳四大名校录取学员超过2000人"

3. 数据支撑
   - "平均提分15-20分"
   - "85%的学员在3个月内见效"
   - "0元试听转化率45%"

4. 风险保障
   - "不满意可退款"
   - "先体验再付费"
   - "专属学习顾问全程跟踪"
```

#### 技能5：异议处理（Objection Handling）

**定义**：理性、专业地回应客户的疑虑和拒绝。

**常见异议及应对**：

```javascript
// 异议1：价格太贵
AI: "我理解您对价格的考量。不过您算一笔账：
- 一节课平均不到30元
- 比起请一对一家教每小时300-500元，我们便宜了10倍
- 而且学习效果有保障，不满意可以退款"

// 异议2：孩子没时间
AI: "我们的课程每天只需15-20分钟，利用碎片时间学习。
- 课程可以随时暂停、回顾
- 不影响孩子的学校作业和其他兴趣班

// 异议3：已经在上其他课程
AI: "多学一个方法不是更好吗？
- 我们的审题四步法可以和其他课程互补
- 相当于给孩子多一套解题工具"
```

**处理原则**：
- ❌ 不要争辩或反驳客户
- ❌ 不要给客户施压
- ✅ 先认同客户感受
- ✅ 再提供解决方案
- ✅ 最后引导客户决策

### 问题1：图片加载失败

**症状**：广告卡片显示占位符而非图片

**排查步骤**：
1. 检查图片URL是否可访问
2. 确认CORS配置是否正确
3. 检查网络连接

**解决方案**：
```javascript
// 检查图片URL
<img 
  src={ad.image} 
  onError={(e) => {
    e.target.src = 'fallback-image.png'  // 设置兜底图片
  }}
  alt={ad.product}
/>
```
