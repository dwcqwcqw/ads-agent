# 广告流量智能系统 - 多智能体协作平台前端

## 🚀 快速启动

### 前端启动

```bash
cd frontend
npm install
npm run dev
```

访问 http://localhost:3000/

### 后端启动

```bash
cd ..
python ad_system.py
```

## 🎨 功能特性

### 1. 现代化UI设计

- **深色主题**：专业的深色配色方案，减少眼睛疲劳
- **毛玻璃效果**：使用 backdrop-filter 实现现代化视觉效果
- **渐变色彩**：丰富的渐变配色方案，提升视觉吸引力
- **响应式布局**：适配各种屏幕尺寸，移动端友好

### 2. 多智能体协作展示

#### 六大智能体工作流

1. **👤 用户画像分析** - 生成详细用户认知模型
2. **🔗 数据补全与增强** - 结合外部数据验证画像
3. **🎯 广告匹配** - 从广告库筛选最佳匹配
4. **🎨 广告创意生成** - 动态生成个性化内容
5. **💬 AI销售对话** - 多轮对话完成转化
6. **📊 优化分析** - 全链路数据建模与优化

#### 交互特性

- ✅ **实时执行动画**：展示智能体执行状态
- ✅ **任务传递动画**：模拟数据在智能体间的传递
- ✅ **可展开详情**：点击智能体查看详细输出
- ✅ **状态追踪**：实时显示执行进度

- **Running**：⚡ 执行中

### 3. 广告推荐系统

#### 推荐算法

基于用户画像，多维度匹配：

1. **需求匹配度** (40%)
   - 广告能否解决用户核心问题
   - 产品功能与用户需求契合度

2. **价格匹配度** (20%)
   - 广告价格与用户消费能力
   - 性价比评估

3. **时机成熟度** (20%)
   - 用户当前购买意向
   - 决策时机判断

4. **兴趣关联度** (20%)
   - 用户历史兴趣标签
   - 潜在需求挖掘

#### 推荐优先级

- 🏆 **Top 1**: 匹配度 94分（教育专项课）
- 🥈 **Top 2**: 匹配度 91分（系统课程）
- 🥉 **Top 3**: 匹配度 88分（硬件产品）
- 🎮 **Top 4**: 匹配度 76分（游戏化学习）
- 🛒 **Top 5**: 匹配度 72分（学习装备）

## 🎯 使用流程

### 1. 查看用户信息

页面加载后，显示输入的用户数据：
- 学生基本信息
- 学习薄弱点
- 家庭背景
- 教育目标

### 2. 启动工作流

点击 **"▶ 开始执行"** 按钮：
- 系统按顺序执行6个智能体
- 每个智能体执行时显示状态
- 任务在智能体间传递时有动画效果

### 3. 查看智能体输出

点击任意智能体，展开查看详细输出：
- 数据分析结果
- 生成的内容
- 性能指标

### 4. 查看广告匹配

工作流完成后，显示：
- 5个匹配的广告
- 包含教育类和游戏、电商类
- 显示匹配度、价格、特征

### 5. 分析优化建议

查看：
- 转化漏斗可视化
- KPI仪表盘
- 优化建议列表
