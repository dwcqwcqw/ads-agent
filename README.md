# 🎯 广告流量智能系统

## 系统概述

云梯构建了一套以**用户认知驱动**的多智能体协作系统，实现用户体验与商业化效率的统一。

### 六大智能体

1. **用户画像分析智能体** - 基于学习数据生成用户认知模型
2. **数据补全智能体** - 结合外部数据源增强用户画像
3. **广告匹配智能体** - 实时识别用户意图并匹配广告
4. **广告生成智能体** - 动态生成个性化广告内容
5. **AI销售智能体** - 多轮对话完成销售转化
6. **优化分析智能体** - 全链路数据建模与反馈优化

## 工作流程

```
用户数据 → 用户画像 → 数据增强 → 广告匹配 → 广告生成 → AI销售 → 优化分析
   ↓          ↓           ↓           ↓           ↓          ↓
 原始输入   认知建模    外部融合    智能推荐   个性化素材   对话成交    持续优化
```

## 快速开始

### 1. 环境要求

- Python 3.8+
- OpenAI API Key
- 网络连接

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 配置API Key

在 `.env` 文件中配置：

```
OPENAI_API_KEY=your-api-key-here
```

### 4. 运行系统

```bash
python ad_system.py
```

## 核心功能

### ✅ 用户画像分析

**输入**：学生学习数据、对话记录、家长需求

**输出**：
- 认知水平评估
- 学习阶段判断
- 核心问题诊断
- 用户需求分析
- 转化意向评估

### ✅ 数据补全

**输入**：用户画像 + 外部数据源

**输出**：
- 数据交叉验证
- 画像增强
- 置信度评估
- 标签完善（行为/兴趣/消费）
- 预测补充

### ✅ 广告匹配

**输入**：增强后的用户画像

**输出**：
- Top 3 最匹配的广告
- 匹配度评分（0-100分）
- 匹配理由
- 预估转化率
- 推荐优先级

**广告池**：
- 教育培训类（数学思维训练营、系统班课）
- 教育硬件类（AI学习机）
- 在线课程类（解题技巧、启蒙课）

### ✅ 广告生成

**输入**：用户画像 + 匹配广告

**输出**：
- 主标题（15字以内）
- 副标题（20字以内）
- 正文文案（50-80字）
- 卖点列表（3-5个）
- **AI生成的广告图片**（使用DALL-E 3）
- 行动按钮文案

### ✅ AI销售

**输入**：广告创意 + 用户画像

**输出**：
- 开场白（建立信任）
- 需求挖掘（3-5轮问答）
- 产品介绍（针对性强）
- 异议处理（价格/效果/信任）
- 成交促成（优惠/承诺）
- 预估成交率

### ✅ 优化分析

**输入**：全链路转化数据

**输出**：
- 转化漏斗可视化
- 关键绩效指标（CTR、落地率、注册率、CPA、ROI）
- 漏斗诊断与归因分析
- 用户群体分析
- 各环节优化建议
- 效果预测

## 系统架构

### 状态管理

```python
MultiAgentState {
    timestamp: str                    # 时间戳
    raw_data: Dict                   # 原始输入
    user_profile: Dict              # 用户画像
    enriched_profile: Dict           # 增强画像
    matched_ads: List                # 匹配广告
    ad_creative: Dict               # 广告创意
    ad_image_path: str              # 图片路径
    sales_dialogue: Dict             # 销售对话
    conversion_data: Dict            # 转化数据
    optimization_report: Dict        # 优化报告
    execution_log: List              # 执行日志
}
```

### 技术栈

- **框架**：LangGraph（状态机工作流）
- **AI模型**：GPT-4o（文本）、DALL-E 3（图片）
- **可视化**：Rich（终端UI）
- **环境管理**：python-dotenv

## 自进化闭环

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│   画像构建 ←→ 数据增强 ←→ 意图匹配                   │
│        ↑           ↑           ↑                   │
│        │           │           │                   │
│   反馈优化 → 内容生成 → 销售转化                     │
│                                                     │
└─────────────────────────────────────────────────────┘
```

系统通过以下方式实现自进化：

1. **数据反馈**：优化分析智能体收集全链路数据
2. **策略调整**：基于转化数据调整各智能体策略
3. **效果评估**：通过KPI监控持续优化
4. **迭代改进**：不断迭代广告匹配和创意生成算法

## 常见问题

### ❓ API Key无效

**解决方案**：
1. 检查 `.env` 文件中的API Key是否正确
2. 确认API Key有足够的调用额度
3. 检查网络连接是否正常

### ❌ 图片生成失败

**解决方案**：
1. 检查DALL-E 3的调用权限
2. 降低图片质量要求
3. 添加异常处理和重试机制

### ⏱️ 运行时间过长

**优化建议**：
1. 减少广告池数量
2. 简化prompt复杂度
3. 添加并行处理机制
